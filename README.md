# The Story Room

> *Your personal sanctuary for words and wonder.*

Live at [thedifferentlove.vercel.app](https://thedifferentlove.vercel.app)

---

## What Is This

The Story Room is a beautifully minimal web app built for readers and storytellers. It gives written content, short stories, poems, quotes, and songs, a home that feels intentional. No clutter. No distractions. Just words on a screen.

The first piece living inside it is **The Different Love**, a short story told in parts. You tap to begin. You read. You feel something.

More content categories are on the way.

---

## Features

- **The Different Love** — a serialized short story, readable in parts
- **Explore section** — a growing library of poems, quotes, and songs (coming soon)
- **Profile section** — personal reading space (coming soon)
- Bottom navigation with Home, Explore, and Profile
- Minimal, distraction-free reading UI
- Mobile-first design

---

## Deployment

Deployed on Vercel. Every push to `main` goes live automatically.

To deploy your own fork:

1. Push your fork to GitHub
2. Import the repo at [vercel.com](https://vercel.com)
3. Set root directory to `/` and deploy

No build command needed. Vercel serves static files directly.
